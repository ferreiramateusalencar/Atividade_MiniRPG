# Contribuindo para MiniRPG

Obrigado por considerar contribuir! Siga estes passos básicos:

- Fork este repositório e crie uma branch com sua feature/fix: `git checkout -b minha-feature`;
- Instale dependências: `composer install`;
- Execute os testes: `vendor/bin/phpunit` (se disponível) e
- Abra um pull request descrevendo sua alteração.

Padrões de código:
- Código em PHP moderno (tipagem, PSR-12 onde aplicável) e
- Testes são bem-vindos para novas funcionalidades e correções.

Se tiver dúvidas, abra uma issue primeiro.
