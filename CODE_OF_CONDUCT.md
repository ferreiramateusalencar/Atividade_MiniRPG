# Código de Conduta

Este projeto segue o Código de Conduta do Contributor Covenant. Ao participar, você concorda em respeitar este documento.

Resumo:
- Seja respeitoso e profissional;
- Evite linguagem ou comportamento discriminatórios e
- Seja receptivo a feedback.

Para mais detalhes, veja: https://www.contributor-covenant.org/pt-br/ .
